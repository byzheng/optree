# optree

This is a lightweight R package for hierarchical runtime configuration. It provides a simple and flexible way to manage nested, multi-level configuration in R, with features such as runtime mutability, type checking, and easy reset to defaults.


## Installation

Currently on [Github](https://github.com/byzheng/optree) only. Install with:

```r
remotes::install_github('byzheng/optree')
```
